# Sortify: Smart File Organizer

A lightweight, standalone Windows app that instantly cleans up your messy folders with zero setup required. Just download, run, and let it do the heavy lifting.

## Features

* **Instant Organization:** Automatically categorizes your files by type (Images, Documents, Videos, etc.) and by Date (Year/Month).
* **Watch Mode:** Click "Start Watch Mode" to let Sortify actively monitor your folder in the background. It will automatically sort any new files the exact second they are downloaded or saved!
* **Smart Hashing:** Uses advanced SHA-256 file hashing to give every file a unique "DNA" fingerprint. It remembers what it has seen to prevent duplicates and skips files that are actively in use.
* **1-Click Undo:** Made a mistake? Click "Undo Last Action" to perfectly reverse your last organizing session. 
* **Dry Run:** Toggle this setting ON to safely test your sorting rules. It will show you exactly what *would* happen in the activity log without actually moving a single file.
* **Permanent Audit Trail:** Keeps a transparent history of every file movement in a beautifully integrated, collapsible activity log (and a permanent `organizer.log` text file).

##  How to Use (No Installation Required!)

Sortify is a portable app. You do not need to install Python or any extra software.

1. Go to the **[Releases](../../releases)** page on this repository.
2. Download the latest `Sortify.exe` file.
3. Place the file anywhere (like your Desktop) and double-click to open it.
4. Click **Browse** to select your messy folder.
5. Click **🚀 Organize Now** for an instant cleanup, or **👁️ Start Watch Mode** to let it organize automatically in the background!
